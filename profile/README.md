# NVIDIA GeForce Game Ready Driver 595.79 (EN) — RTX 30-Series GPUs


## About NVIDIA Driver 595.79

NVIDIA Driver Version 595.79 is an optimized Game Ready driver release for GPUs based on the Ampere architecture, including the GeForce RTX 30-Series. This version brings enhanced performance across modern titles, refined DLSS 3/Frame Generation behavior, improved shader compilation, and updated compatibility for new engines and APIs. It ensures stability, consistent frame pacing, and reliable support for high-refresh-rate gaming, creative workloads, and AI-accelerated applications.

The 595.79 branch focuses on improved scheduling for GPU-bound scenes, better power management under fluctuating loads, stability enhancements for DX12 Ultimate titles, and improved support for ray tracing pipelines.

## Installation Notes

1. Close running games, editors, and GPU-accelerated apps.  
2. Ensure your previous installation is not corrupted (use clean-install if needed).  
3. Run the 595.79 installer and select Express or Custom (Clean Install).  
4. Restart your system once installation completes.  
5. Optionally, configure profiles in NVIDIA Control Panel or GeForce Experience.

## Troubleshooting

- Stutters after update? Rebuild shader cache or enable “Prefer maximum performance”.  
- Black screen after reboot? Use safe mode and reinstall with clean install.  
- DLSS artifacts? Update game profile or disable sharpness filters in NVCP.
